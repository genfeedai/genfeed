## Summary

Generate workflows from plain English descriptions using LLM. Uses LLM to create node graphs based on user intent.

## Key Deliverables

- Natural language input interface
- LLM-powered workflow generation
- Node graph creation from descriptions
- Template suggestions based on intent

## Technical Details

- **AIGeneratorPanel**: Chat interface with user/assistant message bubbles, conversation history for multi-turn refinement, "Load Workflow" button, example prompts for quick start
- **GenerateWorkflowModal**: Content level selector (Empty, Placeholders, Full Content), workflow preview before applying, uses Llama 3.1 70B with temperature 0.3
- **System Prompt Design**: All 15+ node types with inputs/outputs documented, handle IDs for valid connections, layout guidelines, JSON-only output format

## API Endpoints

| Method | Endpoint | Purpose |
|--------|----------|---------|
| POST | `/api/ai/generate-workflow` | Frontend route (Next.js) |
| POST | `/workflows/generate` | Backend route (NestJS) |
