## Summary

Rebrand monorepo from content-workflow to genfeed, change package scope to @genfeedai/*, add AGPL-3.0 license, and prepare for public release.

### Phase 1: Package Renaming

### Phase 2: Licensing

### Phase 4: Self-Hosting

- No secrets in codebase
