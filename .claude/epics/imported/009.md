## Summary

Create a reusable InputGroup component system for grouping form inputs with contextual actions in the workflow editor.

## Scope

- Types in `packages/types/`
- Components in `apps/web/src/components/input-group/`
- Integration with ConfigPanel and nodes

## Acceptance Criteria

- All 4 use cases working: form sections, inline editing, dynamic lists, composite values
- All 4 action UI patterns: hover toolbar, context menu, inline buttons, dropdown
- Keyboard accessible
- Zero external platform dependencies
